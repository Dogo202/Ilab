# Ilab testtask

## Описание
Проект для работы с товарами. Содержит API для добавления товара в базу данных.

## Установка
1. Установите OpenServer.
2. Поднимите локальный сервер через OpenServer.
3. Создайте базу данных, используя SQL-скрипт в папке `sql/database.sql`.

## Использование
- Отправьте POST-запрос на `/product.add.php`, чтобы добавить товар.
- Формат данных: JSON, содержащий информацию о товаре.

## Структура проекта
- `sql/` — SQL скрипты.
- `config/` — Конфигурация базы данных.
- `models/` — Модель продукта.
- `public/` — Точка входа.
